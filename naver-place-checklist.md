## 2. 사진 등록 (매우 중요!)

### 사진 개수 및 종류

**최소 10장 이상 권장**, 카테고리별 다양하게:

#### 외관 (2-3장)
- 건물 전면 (간판 명확히 보이게)
- 주차장 입구
- 건물 입구 안내판

#### 내부 시설 (5-7장)
- 대기실 (깨끗하고 밝은 느낌)
- 운동치료실 (재활 장비 보이게)
- 물리치료실
- 작업치료실
- 진료실 (전문의 책상, 깔끔한 환경)
- 복도 및 화장실 (청결 강조)

#### 치료 장비 (3-4장)
- 보행 훈련 장비
- 근력 강화 기구
- 물리치료 기기 (초음파, 전기치료 등)

#### 치료팀 (1-2장)
- 전문의 및 치료사 단체 사진 (얼굴 보호 처리 선택)
- 환자와 치료사 상호작용 (동의 필수, 모자이크 처리)

### 사진 품질 기준
- **해상도**: 최소 1280 x 720px 이상
- **밝기**: 자연광 또는 적절한 조명
- **각도**: 수평 유지, 왜곡 최소화
- **포맷**: JPG, PNG
- **용량**: 각 파일 5MB 이하

**금지 사항**:
- 과도한 필터나 편집
- 로고나 텍스트 워터마크 (네이버 정책 위반)
- 환자 얼굴 노출 (개인정보 보호법)

## 4. 리뷰 관리

### 리뷰 수집 전략

#### 환자 후기 요청 타이밍
- 초기 치료 4주 후 (효과 체감 시점)
- 치료 목표 달성 시
- 만족도 높은 환자 우선

#### 요청 방법
1. **문자 또는 카카오톡 발송**
   ```
   안녕하세요, 전주E재활의학과의원입니다.
   ○○님의 재활 과정에 함께할 수 있어 기뻤습니다.
   
   치료 경험을 네이버 플레이스에 공유해 주시면
   다른 환자분들께 큰 도움이 됩니다.
   
   [리뷰 작성 링크]
   ```

2. **진료실 안내문** (QR 코드 포함)
3. **대기실 태블릿** (현장에서 즉시 작성 유도)

#### 리뷰 관리 원칙
- **긍정 리뷰**: 감사 댓글 (개인화된 응답)
  ```
  ○○님, 소중한 후기 감사합니다. 
  앞으로도 일상 복귀를 위해 최선을 다하겠습니다.
  - 전주E재활의학과의원 -
  ```

- **부정 리뷰**: 진심 어린 사과 + 개선 의지
  ```
  ○○님, 불편을 드려 죄송합니다. 
  말씀해 주신 부분을 개선하겠습니다.
  개인적으로 연락드릴 수 있을까요? (전화 또는 방문)
  - 전주E재활의학과의원 원장 -
  ```

- **허위 리뷰**: 네이버에 신고 (증빙 자료 첨부)

### 리뷰 평점 목표
- **1년 차**: 평균 4.5점 이상, 20개 이상
- **2년 차**: 평균 4.7점 이상, 50개 이상

## 5. 네이버 예약 연동 (선택)

### 장점
- 환자가 네이버에서 바로 예약 가능
- 노출 우선순위 상승
- 예약 관리 자동화

### 연동 방법
1. 네이버 예약 시스템 신청 (비즈니스 계정)
2. 진료 시간표 설정
3. 예약 알림 설정 (문자/이메일)

### 주의사항
- 예약 변경/취소 정책 명확히 안내
- 노쇼(No-show) 방지 시스템 (사전 확인 전화)

## 6. 네이버 포스트/블로그 연동

- "통원재활 vs 입원재활, 무엇이 다를까?"

- 주 1-2회 (부담 없는 수준)

#### 네이버 플레이스 연동
- 포스트 하단에 플레이스 링크 삽입
- "더 궁금하시면 방문 또는 전화상담 →"

## 체크리스트 요약

### 1단계: 플레이스 등록 (1일)

- [ ] 운영시간, 편의시설 체크
- [ ] 홈페이지, 카카오톡 채널 연결

### 2단계: 콘텐츠 등록 (3-5일)
- [ ] 사진 10장 이상 업로드 (외관, 내부, 장비, 팀)
- [ ] 소개글 작성 (키워드 포함, 500자 이내)
- [ ] 카테고리 정확히 설정

### 3단계: 리뷰 수집 (진행 중)
- [ ] 환자 후기 요청 시스템 구축
- [ ] 긍정/부정 리뷰 응답 매뉴얼 작성
- [ ] 월 5개 이상 리뷰 목표

### 4단계: 콘텐츠 마케팅 (진행 중)
- [ ] 네이버 포스트/블로그 개설
- [ ] 주 1-2회 포스팅 (재활 정보, 건강 팁)
- [ ] 플레이스 연동 및 CTA 삽입

### 5단계: 광고 (선택)
- [ ] 네이버 파워링크 캠페인 설정
- [ ] 플레이스 광고 테스트 (1개월)
- [ ] 전환율 추적 및 최적화
